Modern Calculator Web App

A simple and modern Calculator Application built using HTML, CSS, and JavaScript that allows users to perform basic arithmetic calculations with a clean and interactive interface.
This project demonstrates how to create an interactive calculator using JavaScript logic and responsive UI design.

📌 Project Overview

The Calculator App allows users to perform basic mathematical operations quickly and easily.
Users can click the calculator buttons to perform operations such as:

Addition
Subtraction
Multiplication
Division
The interface updates dynamically as users enter numbers and operations.

✨ Features

🔢 Perform basic arithmetic operations

➕ Addition

➖ Subtraction

✖️ Multiplication

➗ Division

🧹 Clear button to reset the calculation

🎨 Modern UI design

🖱️ Button hover effects for better interaction

📱 Responsive design

🛠️ Technologies Used

HTML5 – Structure of the calculator

CSS3 – Styling and layout design

JavaScript (ES6) – Calculator logic and functionality

⚙️ How It Works

The user clicks the calculator buttons.
Numbers and operators are displayed on the screen.
JavaScript stores the input expression.
When the equals (=) button is pressed, JavaScript evaluates the expression.
The result is displayed on the calculator screen.
